洋裁1,000本ノック | 袖口

この型紙はページをそのまま印刷して利用できます。
A4サイズ100%で印刷してください。
折り伏せ縫いの場合は、破線(5mm)を利用します。
この型紙はパブリックドメインです。個人利用のほか、どのような用途にも無料でかつ無制限に利用できます。

課題

  • 袖口の縫い合わせ（難易度★）

    1. 型紙を実線で切り出す
    2. 型紙に合わせてハギレを裁断する
    3. 身頃と袖を曲線に合わせて布カット幅5mmでロックミシンをかける

  • 袖口の縫い合わせ（難易度★★）

    1. 型紙を実線で切り出す
    2. 型紙に合わせてハギレを裁断する
    3. 身頃と袖を曲線に合わせてぬいしろ10mmで縫う
    4. ぬいしろを5mmカット
    5. ぬいしろにジグザグミシンをかける

  • 折り伏せ縫い(難易度★★★)

    1. 型紙を実線で切り出す
    2. 型紙に合わせてハギレを裁断する
    3. 身頃と袖を中表に合わせてぬいしろ10mmで縫う
    4. 身頃のぬいしろを5mmカットする
    5. アイロンで袖のぬいしろを折り込む
    6. 表側から4mmでステッチをかける

  • 折り伏せ縫い(難易度★★★★)

    1. 型紙を実線と破線で切り出す
    2. 型紙に合わせてハギレを裁断する
    3. 身頃と袖を5mmずらして縫う
    4. 表側から袖のぬいしろを折り込みながら4mmでステッチをかける
